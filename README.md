# Pressure Sensor Course Project

Simple web dashboard for a measuring station.

## What It Shows

- Temperature gauge
- Pressure gauge
- CO concentration gauge
- Live pressure value and status updates
- Pressure status cycles through Low, Moderate, Stable, and High

## How To Run

Open `index.html` in a browser.

No install step is required.

## Main Files

- `index.html` - page structure
- `css/styles.css` - dashboard styling
- `js/pressureSensor.js` - pressure gauge and pressure simulation
- `js/sensorDisplays.js` - temperature and CO gauges
- `js/uiUpdater.js` - updates text and status values in the UI

## Notes

The project uses plain HTML, CSS, and JavaScript. Bootstrap and Font Awesome are loaded from CDNs.
